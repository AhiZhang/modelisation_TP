# 一些有用的库

## `time`库：计算程序的运行时间
### 精度要求不高\程序时间较长
一般使用`time`库以计算程序运行时间
```Python
import time
def main():
    pass

if __name__ == "__main__":
    start_time = time.time()
    main()
    end_time = time.time()
    print(f"It takes {end_time - start_time:.6f}s to run the program")
```
### 精度要求较高\程序时间较短
1. 多跑几次
```Python
import time

def main():
    return

if __name__ == "__main__":
    start_time = time.time()
    for _ in range(10000)
        main()
    end_time = time.time()
    print(f"It takes {end_time - start_time:.6f}s to run the program 100000 times")
```
2. 使用`time.perf_counter()`
```Python
import time

def main():
    for i in range(1000000):
        pass

if __name__ == "__main__":
    start_time = time.perf_counter()
    main()
    end_time = time.perf_counter()
    print(f"It takes {end_time - start_time:.6f}s to run the program")
```

## `math`库：获得常量的值，计算初等函数
### 获取常量
```Python
import math

print(math.pi)
print(math.e)
print(math.inf)
```
### 计算初等函数
```Python
import math

x, y = 5, 3
print(math.pow(x, y))
print(math.exp(x))
```

## `random`库：生成伪随机数
### 在某个区间生成伪随机数
```Python
import random

x = random.randrange(50)       # 生成50(不含)以下的伪随机整数
print(x)
y = random.randrange(0, 100)   # 生成0(含)以上,100(不含)以下的伪随机整数
print(y)
```
### 打乱列表
```Python
import random

l1 = [0, 1, 2, 3, 4, 5]
random.shuffle(l1)             # 打乱列表(原地修改，无返回值)
print(l1)
```

### 多维数组对象
#### 为什么会有所谓的多维数组对象？

可能有人会问,我们既然能够直接使用嵌套列表来实现多维数组,为什么还要用多维数组对象?

必须承认的是,嵌套列表确实能存储多维数组,例如:
```Python
matrix = [[1, 2], [3, 4]]
```
这个列表就能表示矩阵
$$
\begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix}
$$

然而,在处理大规模数据或进行复杂数值计算时,嵌套列表的局限性便会暴露无遗.专门的**多维数组对象**在以下方面有着显著优势:
- 性能
    `NumPy`库是由C语言实现的,运行速度比Python快很多

- 功能
    `NumPy`库支持相当多的数学运算,如矩阵的加减乘,求逆等方法
`NumPy`库的优势还有很多,在此不做赘述
#### 创建多维数组对象`ndarray`
1. 从Python列表\元组直接转换
```Python
# 从列表创建
arr_from_list = np.array([1, 2, 3, 4])
# 从嵌套列表创建二维数组
arr_2d = np.array([[1, 2], [3, 4]])
# 从元组创建
arr_from_tuple = np.array((5, 6, 7))
```
2. 使用内置函数创建
- np.arange
```Python
numpy.arange(start, stop, step)
```
根据`start`(包含)与`stop`(不包含)指定的范围以及`step`设定的步长,生成一个`ndarray`
如:
```Python
# 生成从0到4的数组
x = np.arange(5)
```
- numpy.linspace

```Python
np.linspace(start, end, num=50, endpoint=True)
```
`numpy.linspace`函数用于创建一个等差数列构成的一维数组,其中  
`start`表示起始值  
`end`表示结束值  
`num`表示生成的数量,默认为50  
`endpoint`为`True`时,数列中包含`stop`值,反之不包含,默认为`True`  
如:  
```Python
# 生成1(包含)-10(包含)里有10项的等差数列
x = np.linspace(1,10,10)
# 生成1(包含)-10(不包含)里有9项的等差数列
x = np.linspace(1,10,9, False)
```

- numpy.logspace
```Python
np.logspace(start, stop, num=50, endpoint=True, base=10.0)
```
`numpy.logspace`函数用于创建一个等比数列构成的一维数组,其中  
`start`表示起始值是$base^{start}$  
`end`表示结束值是$base^{end}$  
`num`表示生成的数量,默认为50  
`endpoint`为`True`时,数列中包含`stop`值,反之不包含,默认为`True`  
`base`是取对数时log的下标,默认为10  
如
```Python
x = np.logspace(1.0,  2.0, num =  10)
```



## `matplotlib`库:绘制图像
`Matplotlib`就是专门为`NumPy`数组设计的,即使遇到普通列表,`matplotlib`内部也会先把它转成`numpy`数组再画图
在这里,我们主要介绍`Matplotlib`的子库`Pyplot`,在使用时,我们能够为`Pyplot`设置一个别名`plt`来简化代码:
```Python
import matplotlib.pyplot as plt
```
以下是一些常用的`Pyplot`函数
### `plot`绘制线图
`plot`语法格式如下:
```Python
# 画单条线
plot([x], y, [fmt], *, data=None, **kwargs)
# 画多条线
plot([x], y, [fmt], [x2], y2, [fmt2], ..., **kwargs)
```
