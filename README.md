##  项目简介

本仓库主要用于存储和展示数学建模课程的TP。


## ️ 环境配置

本项目基于 **Python 3** 开发。在运行代码之前，请确保你的环境中已安装 Python。

**克隆仓库**
```bash
    git clone https://github.com/AhiZhang/modelisation_TP.git
    cd modelisation_TP
```



##  文件结构与内容详解

本仓库按模块组织了代码与笔记，结构如下：

### 1. 核心算法 (TP Scripts)

| 文件名 | 核心内容 |
| :--- | :--- |
| **TP_1/** | 复习 |
| **TP_2.py** | 数值计算 |
| **TP_3_recursivite.py** | 递归 |
| **TP_4_tris.py** | 排序 |
| **TP_5.py** | 多项式的运算 |

### 2. 学习笔记 (Notes)

所有的理论推导和课程笔记都整理在 `notes/` 目录中：

- **`notes/modelisation_note.html`**: 格式化的课程笔记，包含数学公式与图表。
- **`notes/modelisation_note.md`**: Markdown 格式的纯文本笔记，便于快速阅读。
